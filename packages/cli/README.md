## Quickly build scaffolding for react projects

- Support

  - react-spa
  - react-components

## How to use

### global download

```code
npm install -g @tiga-cli/cli
```

Then create project

```code
tiga create hello
```

### use npx

```code
npx @tiga-cli/cli create hello
```

Then follow the prompts to create step by step

### command

- create
- init
- build
- serve
- publish
- upgrade
